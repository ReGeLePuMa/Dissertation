This report extends my previous work on container minimization by introducing a novel, 
LLM-driven agentic workflow. By leveraging a Large Language Model to dynamically generate 
targeted fuzzing data, the system actively probes the application to uncover hidden runtime 
dependencies. The report concludes with an evaluation of the results and proposed directions 
for future development.
